# Etihad Airways Brand Fonts

Place the following font files in this directory. Run `npx react-native-asset` or `npx react-native link` to link them.

## Required fonts

| File name (example)     | PostScript name    | Weight   |
|-------------------------|--------------------|----------|
| EtihadAltis-Light.ttf   | EtihadAltis-Light  | Light    |
| EtihadAltis-Book.ttf    | EtihadAltis-Book   | Book (default) |
| EtihadAltis-Text.ttf    | EtihadAltis-Text   | Text     |
| EtihadAltis-Medium.ttf  | EtihadAltis-Medium | Medium   |
| EtihadAltis-Bold.ttf    | EtihadAltis-Bold   | Bold     |
| EtihadArabic.ttf       | EtihadArabic       | Arabic   |

**Fallback:** Arial (system font) when Etihad Altis is unavailable.

Supported formats: `.ttf` or `.otf`
